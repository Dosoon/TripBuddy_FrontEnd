# :computer: 담당한 역할

- 전체 사이트 기획 및 디자인
- 메인 페이지, 마이 플랜, 플랜 생성(편집), TripLog 페이지
  - **컴포넌트** 작성
  - **Axios를 사용한 비동기 API** 호출 및 이후 로직 처리
  - **WebSocket을 사용한 서버와의 실시간 통신 연결 및 로직 처리**
